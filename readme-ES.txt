MÓDULO PRESTASHOP LABSMOBILE SMS

El módulo es compatible con todas las versiones 1.6 y 1.7 de Prestashop

Para instalar este módulo carga el archivo a través de la pestaña Módulos> Instalar módulo y arrastra o selecciona el archivo .zip

Cuando finalice el proceso de subida, haz clic en el botón de instalar y accede a la página principal del módulo.

El primer paso es crear una cuenta y empezar a enviar tus primeros SMS!

/! \ IMPORTANTE /! \

Para un buen funcionamiento del módulo, es imprescindible la introducción obligatoria del número móvil cuando un cliente se registra o se da de alta en tu Prestashop o cuando introduce una nueva dirección.
